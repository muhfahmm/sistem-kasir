# Sistem Kasir "Harmoni"

Proyek Sistem Kasir (Point of Sale) modern yang dibangun dengan arsitektur Fullstack.

## Teknologi Utama
- **Frontend**: Next.js (TypeScript, Tailwind CSS, Lucide Icons)
- **Backend**: Laravel 11
- **Database**: SQLite
- **State Management**: Zustand (Cart & Store)

## Struktur Proyek
- `/frontend`: Aplikasi client Side (POS & Admin Dashboard)
- `/backend`: REST API Server

## Fitur Utama
- [x] Point of Sale (POS) Grid Modern
- [x] Manajemen Produk (CRUD)
- [x] Manajemen Kategori (CRUD)
- [x] Integrasi Real-time dengan Database
- [x] Dark Mode & Mode Terang
- [x] Responsive Design

## Cara Menjalankan
1. **Backend**: `cd backend && php artisan serve`
2. **Frontend**: `cd frontend && npm run dev`
